# FFmpeg

[FFmpeg](https://www.ffmpeg.org/) is a complete, cross-platform solution to record, convert and stream audio and video.

Convert audio and video.
```
ffmpeg -i <input.mp4> <output.avi>
```

## Audio

| Audio options | |
| - | - |
| -an | Disable audio. |

## Video

| Video options | |
| - | - |
| -r | Frame rate. |
